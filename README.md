# Bankamatik Uygulaması

Bu proje, Python pratiği yapmak amacıyla geliştirdiğim temel bir ATM otomasyonudur. 
Kullanıcı giriş yaparak bakiye görüntüleme, para çekme ve para yatırma gibi temel bankacılık işlemleri gerçekleştirilebilir.

## Kullanılan Yapılar
- Dictionary (hesap bilgilerini tutmak için)
- List yapısı
- Fonksiyonlar
- While döngüsü
- Koşul ifadeleri (if-elif-else)
- Try-except ile hata kontrolü
